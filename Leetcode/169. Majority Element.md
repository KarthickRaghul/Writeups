**Question:**

Given an array `nums` of size `n`, return _the majority element_. The majority element is the element that appears more than `âŒŠn / 2âŒ‹` times. You may assume that the majority element always exists in the array.

**Example 1:**

> **Input:** nums = [3,2,3]
> **Output:** 3
**Example 2:**

> **Input:** nums = [2,2,1,1,1,2,2]
> **Output:** 2

**Constraints:**

- `n == nums.length`
- `1 <= n <= 5 * 104`
- `-109 <= nums[i] <= 109`
- The input is generated such that a majority element will exist in the array.

**Optimal Solution**

```java
class Solution {
    public int majorityElement(int[] nums) {
        int res = nums[0] , repeat = 0;
        for(int i:nums){
            if(res==i) repeat++;
            else repeat--;
            if(repeat==0) {
                res = i;
                repeat++;
            }
        }
        return res;
    }
}
```

**Ideology**:
	The maximum repeated element is more then n/2 , therefore when we subtract the remaining elements from it , there will at least one occurrence of it as remaining and we return it as result
